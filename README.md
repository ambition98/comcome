> 주요 서비스
  - 사용자는 원하는 노트북 제품 사양을 필터링하여 검색
  - 해당 제품이 존재하면 제품 설명과 함께 해당 제품을 실제로 판매하는 사이트와 가격을 안내
  - 해당 제품에 대해 의견 등을 작성 가능
  - 해당 사이트로 이동하여 구매 가능

## 1. System Architecture
  - Server: CentOS 8
  - WAS: Tomcat 9.0
  - Backend Language: Java 1.8.0
  - Backend Framework: Spring Boot 2.6.1
  - FrontEnd: JSP 2.3
